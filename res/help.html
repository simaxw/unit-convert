Overview

The SUConvert desktop application is used for converting different type of units into others while typing.

This screenshot shows a list of different unit groups on the left. If one of the groups gets selected the panel on the right will change to that one containing all unit input fields assigned to that particular group.

The toolbar located at the top of the main window contains the following commands:

  • Quit: Terminates the application. Settings are saved to the file settings.ini in the same directory where the executable is located. Shortcut: Ctrl+Q
  • Sort Asc/Desc: Sort the unit groups by name.
  • Previous/Next: Turn to the next/previous unit group respectiveley. Shortcut Ctrl+Up/Down
  • Split/Unsplit: Splits the unit fields and adds another set so that you can convert different values and see the results side by side. Unsplit removes the last set from the right.

Enter Math

You can also enter simple mathematical expressions like 1+2. The result is computed and printed in light gray next to your input.

Since this tool makes use of the JavaScript-like engine QtScript, you can also use the Math object like you would in normal JavaScript.

Example: Math.sin(30*Math.PI/180) calculates the sine of 30 what sould be ~0.5

Also, declaring variables is possible: var x=9; Math.sqrt(x)
